README

This game is "Paper scissors rock",

Make sure all files are same directory which in funkit folder

As you compile, select a character by clicking the controller (WEST or EAST). And then push the controller to send your character to other player. After you send, if other player has not send, the led will display waiting sign, 'w'.

If both two players send their character each other, then the result will be displayed.

-WARINING-
ANY KIND OF INTERFERENCE SUCH AS LIGHT AND SUNSHINE WILL EFFECT THE PROCESS OF GAME
